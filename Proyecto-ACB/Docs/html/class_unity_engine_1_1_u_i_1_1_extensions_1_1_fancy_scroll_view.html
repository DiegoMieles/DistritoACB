Detailed Description

スクロールビューを実装するための抽象基底クラス.

無限スクロールおよびスナップに対応しています. FancyScrollView<TItemData, TContext>.Context が不要な場合は 代わりに FancyScrollView<TItemData> を使用します.

protected

セルを循環して配置させるどうか.

true にすると最後のセルの後に最初のセル, 最初のセルの前に最後のセルが並ぶようになります. 無限スクロールを実装する場合は true を指定します.
