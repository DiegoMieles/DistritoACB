Detailed Description

A leveled logger which could log UniWebView related messages in both development environment and final product.

Member Enumeration Documentation

◆ Level

enum UniWebViewLogger.Level

Logger level.

Enumerator
Verbose 

Lowest level.

When set to Verbose, the logger will log out all messages.

Debug 

Debug level.

When set to Debug, the logger will log out most of messages up to this level.

Info 

Info level.

When set to Info, the logger will log out up to info messages.

Critical 

Critical level.

When set to Critical, the logger will only log out errors or exceptions.

Off 

Off level.

When set to Off, the logger will log out nothing.
