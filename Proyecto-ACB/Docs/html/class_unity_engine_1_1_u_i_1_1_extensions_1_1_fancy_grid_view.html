Detailed Description

グリッドレイアウトのスクロールビューを実装するための抽象基底クラス.

無限スクロールおよびスナップには対応していません. FancyScrollView<TItemData, TContext>.Context が不要な場合は 代わりに FancyGridView<TItemData> を使用します.
