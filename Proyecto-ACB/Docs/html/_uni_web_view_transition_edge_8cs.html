enum UniWebViewTransitionEdge

An enum to identify transition edge from or to when the UniWebView transition happens.

You can specify an edge in Show() or Hide() methods of web view.

Enumerator
None 

No transition when showing or hiding.

Top 

Transit the web view from/to top.

Left 

Transit the web view from/to left.

Bottom 

Transit the web view from/to bottom.

Right 

Transit the web view from/to right.
