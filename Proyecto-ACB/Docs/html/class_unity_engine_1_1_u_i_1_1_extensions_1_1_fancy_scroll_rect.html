Detailed Description

ScrollRect スタイルのスクロールビューを実装するための抽象基底クラス.

無限スクロールおよびスナップには対応していません. FancyScrollView<TItemData, TContext>.Context が不要な場合は 代わりに FancyScrollRect<TItemData> を使用します.
